# Jammer
A matching application for musicians

# Goals
## Minimum Viable Product
* Stack of cards
* Each card has audio clip
* Approve/Deny
* Match
* User Accounts

## Stretch Goals
* Location based matching
* Chat
* Look nice
* Filters (By instrument, distance, etc)
* Audio Clip Uploading
* Account Creation
* Record Audio Clips
* More???
